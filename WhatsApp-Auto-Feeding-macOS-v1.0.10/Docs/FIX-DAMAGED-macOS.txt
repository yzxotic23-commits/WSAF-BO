WhatsApp Auto Feeding — macOS: "damaged and can't be opened"
============================================================

Ini BUKAN file rusak. macOS memblokir app yang:
- diunduh lewat Chrome/Safari (flag quarantine), dan
- belum di-sign Apple Developer.

JANGAN pilih "Move to Trash".

CARA 1 — Terminal (paling ampuh)
--------------------------------
1. Buka Terminal (Applications → Utilities → Terminal)
2. Tempel perintah ini (Enter):

xattr -cr "/Applications/WhatsApp Auto Feeding.app"

3. Buka app lagi dari Applications.

Jika belum di-install, bersihkan DMG dulu lalu install ulang:

xattr -cr ~/Downloads/WhatsApp\ Auto\ Feeding*.dmg

Lalu buka DMG → drag ke Applications → jalankan Cara 1 lagi.

CARA 2 — Klik kanan Open
------------------------
1. Finder → Applications
2. Klik kanan "WhatsApp Auto Feeding" → Open
3. Klik Open di dialog peringatan (bukan Trash)

CARA 3 — System Settings
--------------------------
System Settings → Privacy & Security → scroll bawah
→ "Open Anyway" (muncul setelah percobaan buka pertama)

CATATAN
-------
Untuk distribusi publik tanpa peringatan, perlu Apple Developer
($99/tahun) + notarisasi. Build GitHub Actions saat ini unsigned.
