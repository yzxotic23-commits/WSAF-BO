WhatsApp Auto Feeding — Windows (v1.0.1)
========================================

ISI PAKET (ZIP)
---------------
Folder Installer/
  WhatsApp.Setup.1.0.1.exe   → INSTALLER saja (logo WhatsApp)

Folder Docs/
  PANDUAN-INSTALASI.txt, .env.example, proxies.txt

Setelah install:
  - Aplikasi: WhatsApp.AutoFeeding.exe  (bukan nama Setup)
  - Shortcut desktop: "WhatsApp Auto Feeding"
  - Keduanya berlogo WhatsApp

AUTO-UPDATE (tanpa uninstall)
-----------------------------
1. Di folder install, edit .env:
   APP_UPDATE_URL=https://server-anda.com/wa-updates/
2. Admin: upload isi folder release/updates/ ke URL tersebut
3. App otomatis cek update → banner "New update" → Restart & update

CARA INSTALL
------------
1. Ekstrak semua file dari ZIP ke satu folder.
2. Jalankan Installer\WhatsApp.Setup.1.0.1.exe
3. Ikuti wizard install (bisa pilih folder install).
4. Buka shortcut desktop "WhatsApp Auto Feeding".

SETELAN PERTAMA KALI
--------------------
Setelah install, buka folder data aplikasi (biasanya di folder install):

  - Salin .env.example → .env lalu edit (PAIR_COUNT, OPENAI_AUTH_MODE=codex, dll.)
  - Isi proxies.txt jika pakai proxy (satu baris per proxy)
  - Folder auth/ dibuat otomatis saat connect WhatsApp

CONNECT WHATSAPP
----------------
1. Buka aplikasi.
2. Klik Connect (QR) untuk akun yang ingin di-link.
3. Scan QR di jendela aplikasi dengan HP.

CODEX / AI (opsional)
---------------------
Untuk AI via subscription ChatGPT (tanpa API key):
  - Di PC development: npm run codex-login
  - Token tersimpan di %USERPROFILE%\.codex\auth.json
  - Salin file itu ke PC target, atau login ulang di PC tersebut

FEEDING CHAT PENUH (CLI)
------------------------
Loop AI antar akun masih via terminal di folder project:
  npm start

UNINSTALL (HAPUS APLIKASI)
--------------------------
Cara 1: Windows Settings → Apps → "WhatsApp Auto Feeding" → Uninstall
Cara 2: Start Menu → folder "WhatsApp Auto Feeding" → Uninstall
Cara 3: Folder install → jalankan "Uninstall WhatsApp Auto Feeding.exe"

FITUR DESKTOP
-------------
- Monitor chat per akun (pasangan pair)
- Settings: proxy, .env, Codex login, start/stop feeding
- System log panel

SYARAT PC
---------
- Windows 10/11 64-bit
- Koneksi internet
- Antivirus: jika diblokir, allow "WhatsApp Auto Feeding.exe"

Support: lihat README-DESKTOP.md di folder source project.
